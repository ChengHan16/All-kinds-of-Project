## python_pprint_usage
## 1. 使用 pprint 列印出標準化的輸出
```py
info = {'dog': 1, 'cat': 2, 'elephant': 3, 'alligator': 4, 'ape': 5, 'baboon': 6, 'bear': 7}
print(info)
```

## 2. 使用 pprint 後的 Result
```py
from pprint import pprint
info = {'dog': 1, 'cat': 2, 'elephant': 3, 'alligator': 4, 'ape': 5, 'baboon': 6, 'bear': 7}
pprint(info)
```
